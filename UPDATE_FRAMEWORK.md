## How to update the Bugfish Framework
If you need to update the included Bugfish Framework v.2.75 to a newer available version do following steps:

- Download Bugfish Framework newest version out of here: https://github.com/bugfishtm/bugfish-framework  
- Move all files out of the Bugfish Framework Repositories "_framework" folder into the modules xframework/remote folder. (overwrite files, do not delete files in the modules folder)
- Update files directly on your webserver is the easiest way. Otherwhise feel free to pack  a new zip file and push it into our repository!

Have fun coding!
